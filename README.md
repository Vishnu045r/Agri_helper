Agri Helper:
Agri Helper is a Flutter-based smart agriculture application designed to assist farmers with crop management, agricultural recommendations, and decision support. The application leverages modern mobile technologies and cloud services to provide users with reliable agricultural information and a seamless digital farming experience.

Features:
* Crop recommendation and guidance
* Agricultural information and support
* Firebase cloud integration
* Secure user authentication
* Real-time data synchronization
* User-friendly mobile interface

Tech Stack:
Frontend:
* Flutter
* Dart

Backend & Cloud:
* Firebase Authentication
* Cloud Firestore
* Firebase Storage

Development Tools:
* Android Studio
* Visual Studio Code
* Git
* GitHub
* Flutter SDK
* Firebase Console
